# risk-evaluation-front

## 项目依赖按照
```
yarn install
```

### 开发环境编译和热重载
```
yarn run serve
```

### 生成环境编译压缩
```
yarn run build
```

### 运行测试
```
yarn run test
```

### 代码风格校验并自动解决部分不符合规则的问题
```
yarn run lint
```

### 运行单元测试
```
yarn run test:unit
```

参考 [Configuration Reference](https://cli.vuejs.org/config/).
